# enbolivares-api

Esta api te permite consultar el precio del dólar y el euro en bolívares, utilizando la API de Pydolarvzla.

## Tecnologías Utilizadas

- **Node**
- **Express**
- **TypeScript**
- **API de Pydolarvzla**
